# About Loved

Fresh relationship news, celebrity scandals, and dating trends updated daily.

**Live Site:** [https://aboutloved.com](https://aboutloved.com)

## About

About Loved delivers the latest in relationship news - from celebrity cheating scandals to viral love stories and dating trend analysis. Built with a focus on speed, mobile responsiveness, and real-time updates.

## Features

- **Live News Ticker** - Breaking stories scroll in real-time
- **RSS Aggregation** - Auto-pulls from major entertainment sources
- **Category Filtering** - Scandals, Love Stories, Drama, Advice
- **Mobile First** - Fully responsive design
- **Auto-refresh** - Content updates every 30 minutes
- **Newsletter Signup** - Daily digest subscription

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- GitHub Pages (Hosting)
- Cloudflare (DNS & SSL)
- RSS2JSON API (News feed aggregation)

## Domain

- **Primary:** aboutloved.com
- **Hosted on:** GitHub Pages
- **DNS:** Cloudflare

## Local Development

1. Clone the repo
2. Open `index.html` in browser
3. No build step required

## Deployment

Pushes to `main` branch auto-deploy via GitHub Pages.
